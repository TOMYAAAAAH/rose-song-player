# Rose Song Player

Un lecteur de musique dédié à Rose. Conçu en React

## Fonctionnalités
- Parcourir les sons
- Jouer les sons
- Consulter son historique d'écoute

## Démarrage rapide
1. Clonez le dépôt :  
   git clone https://github.com/TOMYAAAAAH/rose-song-player.git
2. Installez les dépendances :  
   npm install
3. Lancez :  
   npm start

## Projet étudiant
Ce dépôt est un projet étudiant à but pédagogique. Aucune contribution externe n'est acceptée.
